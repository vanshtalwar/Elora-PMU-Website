# Elora PMU Landing Page Website

A mobile-first landing page built to generate leads for Lip Blushing services in Delhi. The focus is on clean design, fast loading, and high conversion.

## ✨ Overview

This project is designed with a premium Korean-inspired aesthetic and a simple user flow. Every section is structured to guide users toward taking action — whether it’s booking, calling, or messaging on WhatsApp.

## 🚀 Features

* Mobile-first responsive design
* Clean hero section with strong CTA
* Before & After showcase
* Benefits and procedure breakdown
* Pricing section (starting from)
* Testimonials for trust building
* Sticky CTA (Call / WhatsApp / Book Now)
* Lead form (Name + Phone)
* WhatsApp integration for quick conversion
* SEO optimized (meta tags, headings)
* Fast loading performance

## 🛠 Tech Stack

* HTML / CSS / JavaScript *(Next.js and Typescript)*
* Deployed on Vercel

## 📈 SEO Focus

* Keyword targeted: **Lip Blushing in Delhi**
* Proper H1, H2 structure
* Meta title & description
* Basic Local Business schema

## 📦 Deployment

The project is live via Vercel for fast and reliable performance.

## 📞 Conversion Focus

The page is built to maximize leads:

* Sticky CTA always visible
* One-click WhatsApp chat
* Simple lead capture form
* Clear pricing entry point
